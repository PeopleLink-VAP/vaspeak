# VASpeak - Project Overview & Status

## What is VASpeak?

**VASpeak** is an English speaking confidence trainer designed specifically for **Vietnamese Virtual Assistants (VAs)**. It's a mini-app site under the `eskills.virtualassistanpro.vn` ecosystem, offering 50+ days of structured lessons to build real-world communication confidence.

---

## Target Audience

- **Primary**: Vietnamese Virtual Assistants looking to improve English speaking confidence
- **Secondary**: Non-native English speakers in customer service/client-facing roles

---

## Core Learning Philosophy

Each day follows a guided **4-block structure**:

| Block | Name | Purpose |
|-------|------|---------|
| 1 | **Listening Decoding** | Train ear to understand native/accented English |
| 2 | **Pattern Drilling** | Repetitive practice of key phrases and structures |
| 3 | **Guided Simulation** | Role-play scenarios with 3 difficulty modes (Practice, Real, Stress) |
| 4 | **Emotional Reflection** | Self-assessment and confidence tracking |

---

## User Progression System

### 4 User Levels
1. **Survival Speaker** - Basic communication survival
2. **Working VA** - Professional daily communication
3. **Client Manager** - Client relationship management
4. **Strategic Partner** - High-level business communication

### Weekly Themes
- Lessons are grouped into weekly themes
- Each week contains 5 daily lessons
- Progressive difficulty within each theme

### Simulation Difficulties
- **Practice** - Low pressure, guided hints
- **Real** - Realistic scenarios, minimal hints
- **Stress** - Time pressure, challenging situations

1. **Completion Rate** - Users completing daily lessons
2. **Confidence Score** - Self-reported improvement over time
3. **Simulation Performance** - Accuracy in guided simulations
4. **Retention** - Users returning for consecutive days

---

## Project Constraints

- **Language**: Primary interface in English, Vietnamese support planned
- **Audio Focus**: Heavy reliance on audio content (TTS, recordings)
- **Offline Support**: Future consideration for low-connectivity scenarios
- **Performance**: Must load quickly on mobile devices
