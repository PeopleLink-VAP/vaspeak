# VASpeak — Design & Branding Guideline

> VASpeak inherits its visual identity from the parent brand **Virtual Assistant PRO** ([virtualassistantpro.vn](https://virtualassistantpro.vn/)), adapted for a mobile-first English learning experience.

## 2. Typography

### Font Stack

| Usage | Font Family | Weights | Source |
|-------|-------------|---------|--------|
| **Headings & Logo** | `Farro` | 300, 400, 500, 700 | Google Fonts |
| **Body Text** | `Roboto` | 300, 400, 500, 600, 700 | Google Fonts |
| **Alternate Headings** | `Montserrat` | 300–700 | Google Fonts |
| **Condensed Labels** | `Roboto Condensed` | 400–700 | Google Fonts |

### VASpeak Adaptation

For the mobile-first learning app, we simplify to:

| Usage | Font Family | Fallback |
|-------|-------------|----------|
| **Headings** | `Plus Jakarta Sans` | `Farro`, sans-serif |
| **Body** | `Inter` | `Roboto`, sans-serif |

> **Rationale**: Plus Jakarta Sans and Inter are already configured in the SvelteKit app (`app.css`) and offer better readability on mobile screens. Farro remains the parent brand's display font.

## 3. Component Styles

### Buttons

| Variant | Background | Text | Border | Radius | Shadow |
|---------|-----------|------|--------|--------|--------|
| **Primary** | `#F2A906` (Sunflower) | `#1B365D` (Navy) | none | `calc(0.5rem - 2px)` | `0 4px 20px rgba(255,215,0,0.3)` |
| **Secondary** | `#1B365D` (Navy) | `#FFFBF1` (Warm White) | none | `calc(0.5rem - 2px)` | `0 4px 20px rgba(27,54,93,0.15)` |
| **Ghost** | transparent | `#1B365D` | `1px solid #1B365D33` | 0.5rem | none |
| **Destructive** | `#990033` | `#FFFBF1` | none | `calc(0.5rem - 2px)` | none |

All buttons use `font-weight: 600`, `padding: 0.75rem 1.5rem`, `transition: all 0.3s ease`.

### Cards

```css
.card {
  background: white;
  border-radius: 16px;           /* generously rounded */
  box-shadow: 0 4px 20px rgba(27, 54, 93, 0.15);  /* navy shadow */
  padding: 1.5rem;
  transition: all 0.3s ease;
}
.card:hover {
  transform: translateY(-4px);
  box-shadow: 0 4px 20px rgba(255, 215, 0, 0.3);  /* sunflower glow */
}
```

### Input Fields

- Background: `white`
- Border: `1px solid #1B365D33` (navy/20)
- Border radius: `0.5rem`
- Focus ring: `2px solid #F2A906` (sunflower)
- Placeholder color: `#1B365D80` (navy/50)

## 5. Animations & Motion

Inherited from VAP's motion system:

| Animation | Duration | Easing | Usage |
|-----------|----------|--------|-------|
| `fade-in` | 500ms | ease-out | Page transitions |
| `float` | 6s | ease-in-out (infinite) | Decorative elements |
| `sway` | 6s | ease-in-out (infinite) | Illustrations |
| `card-hover` | 300ms | cubic-bezier(.4,0,.2,1) | Card lift + glow |
| `spin-slow` | 20s | linear (infinite) | Background accents |

**Motion principle:** Smooth, purposeful animations. No gratuitous motion. Respect `prefers-reduced-motion`.

## 8. Brand Guidelines Summary

| Aspect | Rule |
|--------|------|
| **Primary action color** | Always Sunflower `#F2A906` — never use navy for primary CTAs |
| **Text on sunflower** | Always Navy `#1B365D` — never white on sunflower |
| **Text on navy** | Always Warm White `#FFFBF1` |
| **Background** | Warm White `#FFFBF1` — never pure white `#FFFFFF` for page bg |
| **Cards** | White `#FFFFFF` on Warm White background for contrast |
| **Hover glow** | Sunflower shadow `rgba(255,215,0,0.3)` on interactive elements |
| **Error/destructive** | Crimson `#990033` — never sunflower for errors |
| **Logo font** | Farro (bold 700) for "VASpeak" wordmark |
| **Heading hierarchy** | Navy color, decreasing size, consistent Plus Jakarta Sans |
